## 📱 نصب 


### روش ۱ — استفاده محلی
```bash
git clone https://github.com/MR-Amouei/Panah.git
cd Panah
```

### روش ۲ — دانلود مستقیم
فایل `index.html` را دانلود کنید و بدون هیچ نیازی به سرور، در مرورگر باز کنید.

---

## 🏗️ ساختار

```
Panah/
├── index.html
├── manifest.json
├── sw.js  
```

> تمام منطق و استایل در `index.html` قرار دارند.

---

## 🌓 ویژگی‌های فنی

- **دارک/لایت مود** — با ذخیره تنظیمات و پشتیبانی از `prefers-color-scheme`
- **RTL بومی** — طراحی شده برای زبان فارسی
- **واکنش‌گرا** — موبایل، تبلت و دسکتاپ

## 🤝 مشارکت

اطلاعات اشتباه دیدید یا بیمارستان/پناهگاه جدیدی می‌شناسید؟

1. [یک Issue باز کنید](https://github.com/MR-Amouei/Panah/issues/new) با فرمت:
   ```
   استان: ...
   نام بیمارستان: ...
   آدرس: ...
   تلفن: ...
   ```
2. یا مستقیم Pull Request بفرستید.

## ⚠️ درباره دقت اطلاعات

برخی داده‌ها (شماره‌ها، آدرس بیمارستان‌ها) با کمک هوش مصنوعی جمع‌آوری شده‌اند و ممکن است خطا داشته باشند. اطلاعات تأیید شده با نشان ✓ مشخص هستند.
