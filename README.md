# 🛡️ پناه — راهنمای اضطراری

یک وب‌اپلیکیشن اضطراری برای شرایط بحرانی و جنگی. مستقر روی سرور داخلی، بدون وابستگی به API خارجی، بدون ردیابی.

---

## ✨ ویژگی‌ها

- **داده‌های محلی** — تمام اطلاعات در فایل‌های JSON داخل پروژه نگه‌داری می‌شن، بدون نیاز به API خارجی
- **PWA** — قابل نصب روی موبایل و دسکتاپ مثل یک اپ نیتیو، با پشتیبانی از Service Worker
- **RTL فارسی** — طراحی کامل راست‌به‌چپ با فونت Vazirmatn
- **تم روشن/تاریک** — با ذخیره‌سازی در localStorage
- **ریسپانسیو** — بهینه برای موبایل، تبلت و دسکتاپ

---

## 📂 ساختار پروژه

```
panah/
├── index.html              # فایل اصلی اپلیکیشن

│   ├── hospitals.json      # بیمارستان‌ها

│   └── emergency_numbers.json  # شماره‌های اضطراری
```

## 🗂️ ساختار فایل‌های JSON

### `provinces.json`
```json
{
  "data": [
    {
      "id": 1,
      "status": "published",
      "name": "تهران",
      "capital": "تهران",
      "warning": "⚠️ پرجمعیت‌ترین استان",
      "rc": "هلال احمر تهران — ۰۲۱-۶۶۴۸۰۳۱۱"
    }
  ]
}
```

### `hospitals.json`
```json
{
  "data": [
    {
      "id": 1,
      "status": "published",
      "name": "بیمارستان نجمیه",
      "city": "تهران",
      "address": "خیابان حافظ، تقاطع جمهوری",
      "phone": "021-66707071",
      "verified": true,
      "province": 1
    }
  ]
}
```

### `shelters.json`
```json
{
  "data": [
    {
      "id": 21,
      "status": "draft",
      "description": "۱۱۴ ایستگاه مترو تهران",
      "province": 1
    }
  ]
}
```

### `emergency_numbers.json`
```json
{
  "data": [
    {
      "id": 9,
      "status": "draft",
      "name": "اورژانس",
      "number": "115",
      "icon": "🚑"
    }
  ]
}
```

> فیلد `province` در بیمارستان‌ها و پناهگاه‌ها به `id` در `provinces.json` اشاره می‌کنه.

---

## ➕ افزودن داده جدید

**استان جدید** → `provinces.json` را ویرایش کنید و `id` منحصربه‌فرد بدهید.

**بیمارستان جدید** → `hospitals.json` را ویرایش کنید. مقدار `province` باید برابر `id` استان مربوطه باشد. برای بیمارستان‌های تأییدشده `verified: true` بگذارید.

**پناهگاه جدید** → `shelters.json` را ویرایش کنید و `province` را مشخص کنید.

**شماره اضطراری جدید** → `emergency_numbers.json` را ویرایش کنید.
